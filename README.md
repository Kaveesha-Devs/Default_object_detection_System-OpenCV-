# 🧠 Default Object Detection System - OpenCV

A simple yet functional real-time object detection system built using **Python** and **OpenCV**. This project uses **Haar Cascade Classifiers** to detect faces or eyes through a webcam feed.

## 📌 Features

- ✅ Real-time object detection using webcam
- 🎯 Uses OpenCV's built-in Haar cascade XML files
- 🔁 Modular code for easy extension (e.g., different objects depend ony exm file that used)
- 📂 Organized project structure for collaborative development
